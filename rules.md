# rules.md — כללי הפרויקט (חובה)

## מטרת הפרויקט

אתר “ספר לימוד” בעברית, דפי A4 עקביים, ניווט נוח, הדפסה מושלמת, ועדכון אוטומטי מ־GitHub.

## עקרונות קבועים

- דפי A4 בלבד (מסך + הדפסה).
- ניווט בעברית בכל עמוד: עמוד קודם / עמוד הבא / תצוגת הדפסה / הדפס.
- אין קבצים מיותרים בריפו.
- כל שינוי מבני/עיצובי/תפעולי נרשם כאן לפני פרסום.

## מה נותר לעשות

- להוסיף נושאים נוספים באותו מבנה.
- לשפר גרפיקת “פרימיום” לפי שפה עיצובית שנבחר.

## אתר חי ואייקון קבוע

- האתר מתפרסם אוטומטית דרך GitHub Pages מתוך `/site`.
- כל `git push` מסנכרן את התצוגה החיצונית.
- קיים קיצור דרך על שולחן העבודה בשם **Parabula – ספר לימוד** שמוביל לקישור החי.
- אין לפתוח קבצי HTML מקומיים לצפייה — רק דרך הקישור החי.

## כתיב מתמטי

- הכתיב המתמטי בספר אחיד ומבוסס על MathJax 3.
- כל נוסחה נכתבת ב-LaTeX בלבד.
- בתוך שורה: `\( ... \)`
- נוסחה מוצגת: `

\[ ... \]

`

- אסור להשתמש ב-HTML ידני (sup/sub) לכתיב מתמטי.
- חובה לבדוק מסך + הדפסה (A4) לפני פרסום.
