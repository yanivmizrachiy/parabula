# rules.md — כללי הפרויקט (חובה, מקור אמת יחיד)

## מטרת הפרויקט (במילים חדות)

Parabula הוא **מנוע ספר לימוד דיגיטלי בעברית**:
דפי A4 עקביים למסך ולהדפסה, ניווט נוח, תצוגה חיה דרך GitHub Pages, ותהליך עבודה שמאפשר **עריכה מהירה ומדויקת** לכל עמוד — כולל עבודה מול בינה מלאכותית — בלי לשבור אחידות.

---

## עקרון־על: למה קיימת ההפרדה (כדי שלא יהיו שאלות)

ההפרדה בין HTML ל־CSS והסטנדרטים של A4 קיימים **רק למטרה אחת**:
לאפשר עריכה נוחה ושינויים נקודתיים, תוך שמירה על שפה עיצובית אחידה.

### הגדרה מחייבת (אין פרשנות)
- **Global (קבוע לכל הספר):** דברים שמוגדרים כאן ב־ules.md + קבצי הבסיס (כמו styles/a4-base.css).
- **Per-Page (מותר לשינוי נקודתי):** התאמות של עמוד יחיד בלבד בקובץ styles/pages/עמוד-N.css.
- **אין מצב ביניים**: שינוי שמשפיע על יותר מעמוד אחד = שינוי גלובלי (חייב CHG).

---

## מה קבוע תמיד (Non-Negotiable)

### 1) פורמט ושפה
- עברית מלאה + RTL בכל הממשק.
- הספר מבוסס **דפי A4 בלבד** למסך ולהדפסה.
- חובה לבדוק תוצאה גם במסך וגם בהדפסה לפני פרסום.

### 2) מבנה עמוד A4 (החוק שנאכף בפועל)
בכל דף A4 חייבים להופיע האלמנטים הבאים:
- מעטפת: main.a4-page"; &  param([string]$s) [void]$L.Add($s)  
- כותרת עמוד: .page-title"; &  param([string]$s) [void]$L.Add($s)  
- מטא ל־TOC/נושא: .nav-meta בפורמט **נושא — עמוד X / Y**

> הערה מחייבת: אם דף לא כולל את המבנים האלה — הוא נחשב חריג עיצובית/מבנית.

### 3) הפרדה מוחלטת HTML / CSS (חוק ברזל)
בכל קובצי HTML בפרויקט:
- אסור <style>...</style>"; &  param([string]$s) [void]$L.Add($s)  

העיצוב מגיע רק מקבצי CSS חיצוניים.

### 4) סטנדרט קבצי CSS
- בסיס לכל דפי A4: styles/a4-base.css"; &  param([string]$s) [void]$L.Add($s)  
- בדפי נושאים (pages/* שמייצרים site/*):
  - CSS של הנושא חייב להיות בקובץ style.css של הנושא (או בקובץ CSS ייעודי של העמוד), ולא בתוך ה־HTML.

### 5) כתיב מתמטי (אחיד)
- MathJax 3 בכל הספר.
- LaTeX בלבד:
  - בתוך שורה: \\( ... \\)"; &  param([string]$s) [void]$L.Add($s)  
- אסור שימוש ב־HTML ידני (sup/sub) לכתיב מתמטי.

### 6) ניווט בכל עמוד (מחייב)
בכל עמוד חייב להיות ניווט בעברית שמכיל:
**עמוד קודם / עמוד הבא / תצוגת הדפסה / הדפס**

### 7) תצוגה מקדימה (/preview)
- ב־/preview כפתורי הניווט (מצב תצוגה, הקודם/הבא, וכפתורי נושאים) חייבים להישאר גלויים גם בזמן גלילה.

### 8) אתר חי, קישור קבוע, בלי צפייה מקומית
- האתר מתפרסם אוטומטית דרך GitHub Pages מתוך /site.
- כל git push מסנכרן את התצוגה החיצונית.
- אין לפתוח HTML מקומי לצפייה — רק דרך הקישור החי.
- קיים קיצור דרך קבוע בשם **Parabula – ספר לימוד** שמוביל לקישור החי.

## מה מותר לשינוי נקודתי (Per-Page Allowed)

כל שינוי שמוגדר כאן מותר **רק** בתנאי שלא שוברים שום כלל מהסעיף 

- שינויי תוכן של הדף (טקסט/שאלות/דוגמאות).
- שינויי פריסה פנימית בתוך A4 (כל עוד אין חריגה מגבולות הדף).
- תוספות ייעודיות לעמוד (גרפים/טבלאות/אלמנטים) לפי צורך.
- התאמות עיצוב לדף יחיד **רק** ב־styles/pages/עמוד-N.css.
